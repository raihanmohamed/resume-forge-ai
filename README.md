# ResumeForge AI

ResumeForge AI is a modern AI-powered resume builder web application that helps users create professional resumes quickly and efficiently.

Users can enter personal, educational, and professional details, preview resumes in real time, switch between templates, generate AI-based summaries, and download resumes as PDF documents.

---

## Features

✔ Live Resume Preview

✔ AI Summary Generator

✔ Multiple Resume Templates

✔ Resume Strength Meter

✔ Education & Experience Sections

✔ Local Storage Auto Save

✔ PDF Resume Export

✔ Responsive UI Design
